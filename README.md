# タクシー業務管理Webアプリ デモ版

タクシー事業者向けに、CSV取込から日報、決済集計、カレンダー、経費管理、設定までの画面イメージを確認するための商談・ヒアリング用デモです。
事業者に運用イメージを説明しやすいよう、トップ画面にサービス概要と利用フローを配置し、各画面にデモ表示であることと本実装時の保存対応予定を明記しています。

## 現状仕様

- Next.js App Router / React / TypeScript / Tailwind CSSで実装
- DB、ログイン、サーバー保存、API連携は未実装
- データは `src/lib/demoData.ts` の固定デモデータを使用
- 日報デモデータは `2026-06-24` から `2026-06-30` までの1週間分を持ち、URLクエリまたは日付選択で表示を切り替え
- 1週間分の中に通常営業日、売上が高い日、経費が多い日、営業なし日、CSV未取込日を含める
- CSV取込は実ファイルを保存せず、ファイル名表示、取込プレビュー、取込完了状態のみ表示
- 添付CSV仕様を想定し、F1営業管理、F2出庫、F3入庫、F4営業時系列、FA決済データを読み取ったようなデモ表示
- CSV取込から日報生成、決済集計、カレンダー確認、経費管理までの利用フローをトップ画面で説明
- カレンダーの日付セルから `/daily-report?date=YYYY-MM-DD` 形式で日報へ遷移
- ダッシュボードの今月売上はカレンダーの今月合計と同じ集計値を表示
- 決済集計は日次、週次、月次、期間指定で集計金額、件数、構成比、明細件数を切り替え
- 決済種別クリックで利用明細を表示し、突合済み、差額あり、未入金の突合ステータスを表示
- 決済明細の突合ステータスと備考は、画面上だけで手動編集できる
- 決済明細は、すべて、突合済み、差額あり、未入金、未入金・差額ありで絞り込み可能
- 絞り込み件数はステータス編集後の画面状態をもとに再計算
- 各画面に「これはデモ表示」「本実装ではCSV/DB保存対応予定」が伝わる注記を表示
- PC表示を優先し、タブレット幅でも横スクロールや折り返しで崩れにくい構成
- 画面上に「デモ版」バッジを表示

## 実装画面

- ダッシュボード: サービス概要、利用フロー、本日売上、今月売上、営業回数、キャッシュレス比率、経費合計、差引利益、最近取り込んだCSV、決済内訳、月間売上推移
- CSV取込: ファイル選択、ファイル名表示、LT27 CSVのF1/F2/F3/F4/FA説明、取込プレビュー、解析結果風カード、取込済み一覧、完了メッセージ
- 日報: URLクエリ対応、日付切り替え、未登録日の空状態、CSV未取込日の専用表示、帳票風ヘッダー、乗務員コード、車両コード、出庫/入庫時刻、営業回数、売上、決済内訳、経費、利益、営業明細、CSV/PDF/印刷ボタン
- カレンダー: デモ注記、日報への日付リンク、月間カレンダー、売上、経費、利益、CSV未取込日、営業なし日、売上高めの日、週/月/前月比サマリー
- 決済集計: デモ注記、日次、週次、月次、期間指定のタブ風UI、決済種別別の金額、件数、構成比、バー表示、決済種別クリック時の利用明細、ステータス絞り込み、横スクロール対応テーブル、突合ステータス選択、備考編集
- 経費管理: デモ注記、今月経費、ガソリン代、駐車場代、その他経費のサマリー、1週間分の日報・カレンダーと金額感を合わせた経費一覧、ラベル付き追加フォーム風UI、画面上だけの一時追加、領収書アップロードボタン
- 設定: デモ注記、会社情報、乗務員、車両、決済種別マッピング、QR/NETはデモ用分類である注記、CSV取込ルール

## ダミー実装

- CSV解析、ファイル保存、サーバー保存は行いません。
- 日報CSV出力、PDF出力、印刷、決済CSV出力、領収書画像アップロード、設定保存は `alert` のみです。
- 経費追加はブラウザ上の一時的な状態反映のみで、リロードすると戻ります。
- 決済明細の突合ステータス変更、備考編集、ステータス絞り込みはブラウザ上の一時状態のみで、リロードすると戻ります。
- 突合ステータスと備考はデモ表示です。本実装では入金データとの照合結果をDB保存する想定です。
- 日付切り替えは固定デモデータの範囲でのみ表示が変わります。CSV未取込日は専用表示、範囲外の未登録日は空状態を表示します。
- 画面内の注記は商談・ヒアリング時に本実装範囲を説明するためのものです。

- LT27 CSV仕様に沿った実CSVパーサー
- 取込履歴、日報、経費、設定のDB保存
- 認証、権限管理、事業者ごとのデータ分離
- 日報PDF/CSVの実出力
- 決済種別マッピングの編集保存
- 領収書画像のアップロードと経費明細への紐付け
